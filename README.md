# Applicazione CRUD con Laravel

Semplice CRUD realizzato con Laravel, che permette di creare, leggere, aggiornare e cancellare utenti.

## Avvio del progetto in locale

Per lo sviluppo locale, il progetto fa uso di Sail, perciò è necessario installare Docker.

Una volta installate le dipendenze con composer, digitare:

```bash
vendor/bin/sail up
```

per avviare il progetto. L'applicazione sarà disponibile all'indirizzo http://localhost

## Test

Per lanciare i test, digitare: 

```bash
vendor/bin/sail artisan test
```
